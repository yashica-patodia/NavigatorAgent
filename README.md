# How to Use the Agent
1. Visit the link above to access the SeedToScale Navigator
2. Type your question in the search box about startup growth, fundraising, product
development, or any other startup-related topic
3. Press Enter or click "Search" to submit your query
4. Wait a moment while the agent searches across Accel's SeedToScale knowledge base
5. Review the response which includes:
  ○ A detailed answer to your question
  ○ Citations from relevant SeedToScale articles
  ○ Source links to read the original content

# Example Questions to Try
1. "What advice does SeedToScale provide for Series A fundraising?"
2. "How should I approach product-market fit?"
3. "What are effective go-to-market strategies for B2B SaaS?"
4. "How do I build a strong founding team?"
5. "What metrics should I focus on for my early-stage startup?"

# Technical Notes
1. The agent uses AI to understand your questions and find relevant information
2. All responses are based exclusively on content from Accel's SeedToScale platform
3. The search is powered by vector embeddings for semantic understanding
4. Responses are generated using ChatGPT to provide natural, helpful answers
